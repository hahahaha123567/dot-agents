## 功能

为文章添加宏观分类标签，用于大致区分文章类别。

## 宏观分类标签

每个文章只需要 1-2 个宏观标签。

**常见宏观标签：**
- #游戏
- #娱乐
- #科技
- #经济
- #政治
- #社会
- #历史
- #文化
- #教育
- #健康
- #其他

## 使用方法

1. 读取文章内容
2. 根据文章主要内容主题选择最相关的一个宏观标签
3. 如果文章涉及多个领域，选择最重要的那个
4. 将标签添加到文章开头的原始链接下方

## 文件格式要求

文章开头格式：
```markdown
[文章标题](原文链接)

#宏观标签1 #宏观标签2

#文章来源 #来源类型

文章内容
```

## 示例

原文：
```markdown
[一篇远古文章：谁谋杀了我们的游戏？ - 知乎](https://zhuanlan.zhihu.com/p/200925087)

# 01 引文
...
```

添加标签后：
```markdown
[一篇远古文章：谁谋杀了我们的游戏？ - 知乎](https://zhuanlan.zhihu.com/p/200925087)

#游戏 #娱乐

# 01 引文
...
```
